# Face Detection & Recognition System
### Minor Project BTP2CSE289 | IILM University Greater Noida

## About
Real-time face detection and recognition system built
using Python and OpenCV LBPH algorithm. Detects and
recognizes faces through live webcam with confidence
score display.

## Technologies Used
- Python 3.14
- OpenCV (opencv-contrib-python)
- NumPy, Pillow

## How to Run
pip install opencv-contrib-python numpy Pillow

python dataset_creator.py  
python trainer.py           
python recognizer.py        

## Features
- Haar Cascade face detection
- LBPH face recognition
- Real-time confidence score display
- Green box = recognized, Red box = unknown
